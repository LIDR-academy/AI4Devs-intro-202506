# Prompts utilizados

Se utilizó el chatbot https://claude.ai/ con el siguiente prompt:

> Crea una página web con las siguientes características:
>
> - Utilizar la plantilla "index.html" (archivo adjunto).
> - Utilizar la imagen adjunta "reversestring-example.png" de referencia visual.
> - Agregar lógica en javascript que invierta el orden de una cadena de texto. Ejemplo: si introduzco AI4Devs devuelve sveD4IA.
>   Retorná las siguientes salidas:
> - html de la página
> - javascript de la página

> [!NOTE] Nota
> Se adjuntaron los archivos:
>
> - index.html
> - reversestring-example.png
> - script.js
