# Prompts utilizados para dar con el código final

Chatbot utilizado: Gemini 3 (Razonamiento)

1. Prompt inicial
    ```markdown
    # Crea una página web con lógica en javascript que invierta el orden de una cadena de texto

    ## Stack
    - JS vainilla
    - CSS

    ## Requisitos funcionales

    - Interfaz dinámica, amigable y moderna
    - Campo de texto para introducir la cadena, sin necesidad de botón de envío
    - Cálculo de reversión de cadena automático al introducir el último caracter en el formulario.

    ## Requisitos técnicos

    - Separación de lógica y presentación en ficheros externos JS y CSS respectivamente
    - Uso del último estándar de JS (ECMAScript) adoptado por la mayoría de navegadores
    - CSS3
    ´´´
