Prompt 1: Solicitud inicial
text
Necesito crear una página web con HTML y JavaScript que invierta el orden de una cadena de texto.
Requisitos:
⦁	Debe tener un campo de entrada (input) donde el usuario pueda escribir texto
⦁	Un botón para ejecutar la inversión
⦁	Un área donde se muestre el resultado
⦁	El JavaScript debe estar en un archivo separado llamado script.js
⦁	El HTML debe estar en un archivo llamado index.html
Ejemplo de funcionamiento:
⦁	Entrada: AI4Devs
⦁	Salida: sveD4IA
Por favor, genera ambos archivos con código limpio y comentado.
