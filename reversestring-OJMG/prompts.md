# Prompt 1

Quiero hacer una página web html que tenga un input para capturar un string con una longitud mínima de 3 caracteres y un máximo de 100 caracteres, debajo de ese input quiero que pongas una caja de texto donde se visualice el texto introducido en el input haciendo un reserve string.

El reverse string debe ejecutarse con un botón.

Usa el siguiente html como plantilla base:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Reverse String</title>    
</head>
<body>
<script src="script.js"></script>
</body>
</html>

Y gener el contenido javascript necesario en un solo archivo script.js
