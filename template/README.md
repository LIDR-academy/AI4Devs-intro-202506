# Inversor de Texto (Página HTML + JS)

Esta es una página web sencilla en **HTML5** que permite invertir cualquier texto ingresado por el usuario.

## ¿Cómo usar?
1. Abre `index.html` en tu navegador.
2. Escribe un texto en el campo de entrada.
3. Presiona el botón **"Inventir texto"**.
4. El resultado invertido se mostrará en la etiqueta junto a “Resultado:”.

## Contenido del proyecto
- `index.html`: Estructura de la página y elementos de la interfaz.
- `app.js`: Lógica para invertir el texto y enlazar eventos de la UI.
- `ALGORITMO.md`: Documento técnico explicando el algoritmo de inversión.
- `README.md`: Este documento con instrucciones de uso y overview.

## Validación de ejemplos
Al cargar la página, se realiza una validación automática en memoria con los ejemplos solicitados:

- `Hormiga` → `agimroH`
- `Amarillo` → `olliramA`
- `Banco` → `ocnaB`
- `Rastrillo` → `ollirtsaR`

Si todo es correcto, verás un mensaje “Validación automática: todos los ejemplos fueron invertidos correctamente.”  
Cualquier discrepancia se mostrará en la consola del navegador (F12 → Console).
