



Crea una **página web con lógica en javascript que invierta el orden de una cadena de texto**.

Ejemplo: si introduzco `mañana es halloween` devuelve `neewollah se anañam`.

Hazlo apoyado en el seed `index.html` que proporcionamos dentro de la carpeta template

# template

Encontraras adjunto una imagen con el template definido para el diseño de la pagina, este no debe ser modificado.

#index.html

<!DOCTYPE html>

<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Reverse String</title>  
</head>
<body>
<script src="script.js"></script>
</body>
</html>
