Actúa como un **experto en desarrollo de páginas web**. Tu tarea es **generar el código necesario para una página web** que cumpla con los siguientes requisitos:

- La página debe ser un único archivo llamado `index.html`.
- Debe incluir:
  - Una **caja de texto** donde el usuario pueda escribir una cadena.
  - Un **botón de aceptar**.
- Al pulsar el botón, la página debe:
  - **Revertir la cadena introducida** en la caja de texto.
  - Mostrar el resultado en la misma página, sin recargarla.
- El código debe ser **limpio**, **modular** y estar **comentado** para facilitar su comprensión.

Entrega únicamente el código completo de `index.html`.
