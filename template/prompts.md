# 📋 Prompt Mejorado para Experto en Desarrollo Web

Contexto: Necesito una aplicación web funcional que invierta cadenas de texto en tiempo real.
Requerimientos Técnicos:

## Funcionalidad principal

Crear una página HTML con un formulario que contenga un campo de entrada (input)
Implementar inversión de string en tiempo real (mientras el usuario escribe)
La inversión debe procesar cada carácter del string de derecha a izquierda
Ejemplo: Input "AI4Devs" → Output "sveD4IA"

## Especificaciones de implementación

HTML semántico y accesible (usar labels, atributos ARIA si es necesario)
JavaScript vanilla (sin frameworks/librerías externas)
Código limpio, comentado y siguiendo mejores prácticas
Responsive design básico
Manejo de edge cases (string vacío, caracteres especiales, emojis, etc.)

## Experiencia de usuario

Actualización inmediata del resultado mientras el usuario escribe (evento input)
Feedback visual claro del resultado
Interfaz intuitiva y minimalista

## Requisitos de entregable

Archivo único HTML auto-contenido (CSS y JS inline o en tags internos)
Compatible con navegadores modernos (Chrome, Firefox, Safari, Edge)
Sin dependencias externas

## Consideraciones Adicionales

Validar que se preserve la integridad de caracteres Unicode
Código eficiente (evitar operaciones costosas en cada keystroke)
Opcional: agregar animación sutil o transición al mostrar resultado

Formato de Entrega: Artifact tipo html junto con javascript con estructura completa y funcional.
