
Crea una página web con lógica en javascript que invierta el orden de una cadena de texto. Con los siguientes requisitos
- La cadena de texto tiene que invertirse cuando el usuario escribe. 
- Tiene que haber un boton de copiar con feedback para el usuario
- TIene que verse como la imagen adjunta
Ejemplo: si introduzco AI4Devs devuelve sveD4IA.
Hazlo apoyado en el seed index.html y script.js que proporcionamos dentro de la carpeta template
El template esta en este repo https://github.com/LIDR-academy/AI4Devs-intro-202506
