## Instrucciones
-Desarrolla una aplicación web que permita al usuario introducir una cadena de texto, seleccionar un botón y recibir la cadena de texto invertida.
-La aplicación debe contener una pequeña cabecera con información para el usuario de la función que cumple la aplicación y el objetivo de la misma.
-El usuario debe poder introducir en un input la cadena de texto y debe existir un botón con el siguiente texto: "Reverse" para que cuando el usuario lo seleccione la aplicación ejecute la lógica correspondiente para devolver la cadena de texto inversa al usuario
-El usuario debe poder copiar la cadena de texto con un botón que debe aparecer justo lado de la cadena de texto invertida
-El estilo y diseño web debe seguir estandares web modernos y debe ser un diseño atractivo y actual
-En la cabecera de la aplicación a la derecha, debe aparecer con un botón la opción para que el usuario pueda cambiar el tema de la aplicación de claro/oscuro y a la inversa

## Reglas y convenciones
-Los estilos de la aplicación deben estar introducidos en el mismo html dentro de un apartado </style>
-No deben generarse estilos en línea. Siempre deben crearse clases CSS y usar las variables definidas para los colores, fuentes y demás aspectos de diseño
-Deben definirse estilos responsive para adaptar la web a distintos dispositivos
-Las classes CSS deben seguir kebab-case
-Variables y funciones en Javascript camelCase
-Clases y constructores en Javascript PascalCase
-Constantes en Javascript UPPER_CASE
-Utilizar arrow functions
-Documenta en JSDOC en idioma español toda la lógica desarrollada
