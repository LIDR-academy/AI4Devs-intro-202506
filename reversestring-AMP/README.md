# PoC: Reverse String

Este proyecto es una Prueba de Concepto (PoC) que consiste en una página web simple para invertir una cadena de texto introducida por el usuario.

##  Funcionalidades

-   **Entrada de texto**: Un campo para que el usuario escriba o pegue cualquier cadena de texto.
-   **Inversión de texto**: Un botón "Reverse" que, al hacer clic, muestra la versión invertida del texto.
-   **Copiar al portapapeles**: Un botón "Copy" que aparece con el resultado y permite copiarlo fácilmente.
-   **Diseño responsivo**: La interfaz se adapta a diferentes tamaños de pantalla.

## Tecnologías Utilizadas

-   **HTML5**: Para la estructura y el contenido de la página web.
-   **CSS3**: Para el diseño y la apariencia visual, integrado en el archivo HTML.
-   **JavaScript (ES6+)**: Para toda la lógica interactiva, incluyendo la manipulación del DOM, la inversión del string y la funcionalidad de copiado.

## ¿Cómo ejecutarlo en local?

No se necesita ningún servidor web ni dependencias complejas. Sigue estos sencillos pasos:

1.  **Descarga los archivos**: Asegúrate de tener los dos archivos, `index.html` y `script.js`, en la misma carpeta.
2.  **Abre el archivo HTML**: Haz doble clic en el archivo `index.html` o arrástralo a tu navegador web preferido (como Chrome, Firefox, Edge, etc.).
3.  **¡Listo!**: La aplicación se cargará en el navegador y podrás empezar a usarla.
