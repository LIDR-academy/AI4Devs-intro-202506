🛠️ Reversal Methods Used

Method 1: split() + reverse() + join()

The most common and readable approach:

string.split('').reverse().join('')

Splits the string into an array of characters, reverses the array, then joins back into a string.

Method 2: For Loop (Backward)

Traditional loop approach for better performance:

for (let i = str.length - 1; i >= 0; i--)

Iterates through the string from the last character to the first, building the reversed string.

Method 3: Recursion

Elegant functional programming approach:

str.charAt(str.length-1) + reverse(str.substring(0, str.length-1))

Takes the last character and recursively reverses the rest of the string.
