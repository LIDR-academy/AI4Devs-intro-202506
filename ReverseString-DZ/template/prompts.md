# Prompts Used

## Prompt 1
Create a simple web page using HTML and JavaScript that allows a user to input a string and reverse it when clicking a button.

## Prompt 2
Improve the solution to correctly handle Unicode characters and emojis when reversing a string.

## Final Approach
Used JavaScript function:
Array.from(text).reverse().join('')

Reason:
This handles Unicode characters more reliably than split('').
