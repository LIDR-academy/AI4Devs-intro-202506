## Task
Create script.js, it should contain the necessary code to reverse a input string.
## Constraints
Make it efficient and optimized.
If the string has 3 or more characters: 
	It will start reversing the string in real time with a animation.
	A optional "reverse string" button will appear. 
	A new section "Your reversed string is: " will appear if the optional button is clicked and will contain the reversed string in bold font.
## Input
index.html
```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Reverse String</title>    
</head>
<body>
<script src="script.js"></script>
</body>
</html>
```
