# Prompts

## Prompt 1

Necesito que generes una página web con lógica en JavaScript que invierta el orden de una cadena de texto. Ejemplo: si introduzco `AI4Devs`, la aplicación debe devolver `sveD4IA`.
Puedes partir de los dos archivos adjuntos como plantilla.
Como layout, incluye el título Reverse String destacado, utiliza una línea para el campo de texto de entrada, y otra línea debajo para ir mostrando el resultado en tiempo real conforme el usuario vaya insertando el texto. Procura que el diseño sea claro y elegante, minimalista.
Si tienes alguna duda o detalle que clarificar, preguntame antes de empezar a generar el código
