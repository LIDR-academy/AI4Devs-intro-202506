# Prompts utilizados

## Prompt 1
Crea una página web con JavaScript que invierta una cadena de texto ingresada por el usuario.

## Prompt Final
Genera una solución para una prueba técnica llamada "Reverse String".  

Debes crear el contenido de dos archivos: `index.html` y `script.js`, partiendo de un template donde `index.html` ya tiene la estructura básica y la referencia al archivo `script.js`.  

### Requisitos funcionales:
- En `index.html`, agrega:
  - Un campo de entrada de texto (`input`).
  - Un botón para ejecutar la acción.
  - Un elemento donde se muestre el resultado.
- Al hacer clic en el botón:
  - Obtener el texto ingresado por el usuario.
  - Invertir la cadena (ejemplo: `AI4Devs` → `sveD4IA`).
  - Mostrar el resultado en la misma página sin recargar.

### Requisitos técnicos:
- Usar únicamente HTML, CSS básico (opcional) y JavaScript puro (vanilla JS).
- No usar frameworks ni librerías externas.
- Toda la lógica debe estar en `script.js`.
- Usar `addEventListener` para manejar eventos.
- Manipular el DOM correctamente.
- El código debe ser claro, limpio y con comentarios breves.

### Consideraciones:
- Manejar el caso en que el input esté vacío.
- Usar una solución eficiente para invertir la cadena (por ejemplo: `.split('').reverse().join('')`).
- Usar nombres de variables claros.
