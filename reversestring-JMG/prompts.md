# Aplicación Web para Invertir Cadenas de Texto
Estamos construyendo una simple aplicación web del lado del cliente que permite al usuario introducir una cadena y ver la versión invertida de esa cadena en pantalla.  
El objetivo es implementar esto con una clara separación entre la estructura HTML y la lógica JavaScript. El proyecto está estructurado usando una carpeta `template` con los archivos `index.html` y `script.js` proporcionados.

## Stack Tecnológico:
- HTML5  
- JavaScript (Vanilla, ES6)  
- CSS (opcional, para estilización básica)

## Requisitos Funcionales:
- Renderizar un formulario básico que incluya:
  - Un campo de entrada donde el usuario pueda escribir una cadena
  - Un botón etiquetado como "Invertir texto"
  - Un contenedor o etiqueta donde se mostrará la cadena invertida

- La aplicación debe:
  - Leer la cadena introducida cuando se hace clic en el botón
  - Invertir la cadena usando lógica de JavaScript
  - Mostrar el resultado invertido debajo del campo de entrada

## Estructura de Archivos:
- `template/index.html`: Contiene la estructura de la página (formulario y contenedor para el resultado)
- `template/script.js`: Contiene la lógica para capturar el evento del botón, invertir la cadena y renderizar el resultado
