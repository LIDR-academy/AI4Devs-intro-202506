Quiero una página web en JAVASCRIPT en la que un usuario agregue en un input una cadena de texto y de forma automática podamos ver los caracteres que va insertando en forma inversa

por ejemplo: user resu

Necesito que la UI esté en un archivo index.html y la lógica del código en script.js

# 2

Crea una página web sencilla usando **HTML, CSS y JavaScript puro**.

### Requisitos

* La interfaz debe estar en un archivo llamado **index.html**.
* La lógica de JavaScript debe estar en un archivo separado llamado **script.js**.
* En la página debe haber:

  * un **input de texto**
  * un área donde se muestre en tiempo real el texto escrito por el usuario **en orden inverso**
* El comportamiento debe ser automático mientras el usuario escribe, sin necesidad de pulsar ningún botón.
* Ejemplo:

  * si el usuario escribe: `user`
  * debe mostrarse: `resu`

### Condiciones técnicas

* Usa **JavaScript vanilla**, sin frameworks ni librerías.
* Conecta correctamente `script.js` desde `index.html`.
* El código debe estar limpio, bien organizado y fácil de entender.
* Añade una estructura HTML básica y una interfaz visual simple pero clara.
* Incluye comentarios breves en el código para explicar la parte principal de la lógica.

### Entrega esperada

Devuélveme el contenido completo de:

1. `index.html`
2. `script.js`

Asegúrate de que funcione al abrir `index.html` en el navegador.
